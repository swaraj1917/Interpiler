How to use Interpiler

1
Write rough code — don't worry about being perfect. Even broken code is useful!
2
Click Run — Interpiler will try to understand your code.
3
Watch the pipeline — see exactly which stage (Lexer → Parser → AST → Interpreter) is working and what it's doing.
4
Read the feedback — errors are explained in plain English, and you get tips to fix your code.

Quick syntax: x = 5  ·  print x  ·  if (x > 0) { print x }  ·  while (x < 5) { x = x + 1 }  ·  for (i=0; i<5; i=i+1) { print i }
